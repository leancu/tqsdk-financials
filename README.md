## 策略详情

### 01 — 沪深300股指期货趋势策略 (`01_if_trend.py`)

- **合约**：`CFFEX.IF2503`（沪深300股指期货）
- **框架**：大周期（日线）过滤趋势方向 + 小周期（小时线）精确入场
- **趋势判断**：日线 EMA20，收盘价上方为多头，下方为空头
- **入场信号**：小时线 MACD(12,26,9) 金叉做多 / 死叉做空
- **风控**：止损 0.5%，止盈 1.0%（风险回报比 1:2），收盘前强制平仓
- **适用场景**：趋势行情，避免震荡市频繁交易

### 02 — IC/IF价差对冲策略 (`02_ic_if_spread.py`)

- **合约**：`CFFEX.IC2503`（中证500）与 `CFFEX.IF2503`（沪深300）配对
- **核心原理**：IC/IF 归一化价差 `(IC-IF)/IF×100%` 历史均值回归
- **入场信号**：价差 Z-score > 1.5σ 做空价差（空IC多IF）；< -1.5σ 做多价差（多IC空IF）
- **平仓信号**：Z-score 回归至 ±0.3σ 区间内平仓；超过 ±2.5σ 强制止损
- **适用场景**：大小盘风格轮动行情，市场中性策略

## 使用说明

1. 安装依赖：`pip install tqsdk numpy pandas`
2. 在策略文件中替换 `TqAuth("sim_user", "sim_password")` 为你的天勤账户
3. 回测：取消注释 `TqBacktest(...)` 行，注释掉模拟盘行
4. 实盘：取消注释 `TqAccount(...)` 行，填入期货账户信息
